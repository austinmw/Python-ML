# Python Machine Learning - Supplementary Datasets

## Iris Flower Dataset

- Used in chapters 1, 2, and 3

The Iris dataset for classification.

**Features**

1. Sepal length
2. Sepal width
3. Petal length
4. Petal width

- Number of samples: 150

- Target variable (discrete): {50x Setosa, 50x Versicolor, 50x Virginica}
